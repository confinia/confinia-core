Confinia

An EU-hosted API for administrative boundaries with full historical versioning — so European developers get GDPR-clean location data without Google Maps' pricing or the compliance headaches of a US-based provider.

🗺 Live demo — boundaries through time 📖 API documentation ⚡ API examples 💶 Pricing 📓 Journal

Quickstart

# Which French commune was here on 2015-06-01? (codes get reused, names change, communes merge)
curl "https://api.confinia.io/v1/communes?lat=46.11&lon=5.83&at=2015-06-01"

# Any European municipality, any date — 43 countries, 168k historical versions
curl "https://api.confinia.io/v1/units?lat=52.52&lon=13.405&at=2020-06-01"   # Berlin
curl "https://api.confinia.io/v1/units?lat=41.39&lon=2.17&at=2020-06-01"     # Barcelona

# Full history of one unit
curl "https://api.confinia.io/v1/communes/01033/history"

# NUTS regions, 7 versions (2003→2024)
curl "https://api.confinia.io/v1/nuts?level=3&country=FR&at=2022-06-01"

Free during development — no key required yet. For the beta, grab a key: POST /v1/keys {"email": "you@example.org"} and send it as X-API-Key.

Coverage

  • France — communes at exact INSEE event dates since 1943 (counts match INSEE published figures exactly), IGN geometries, mergers/splits genealogy.
  • Germany — Gemeinden 2016→2025 (BKG VG250 yearly editions).
  • Netherlands — gemeenten 2016→2026 (CBS/Kadaster yearly editions).
  • All of Europe — municipalities (Eurostat LAU 2016→2023) and NUTS regions (7 versions, 2003→2024), 43 countries.
